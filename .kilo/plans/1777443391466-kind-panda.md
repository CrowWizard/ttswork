# Usage Code TTS Limit Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use trycycle-executing to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将文本转语音的免费生成限制为匿名用户和注册用户各 1 次，免费生成单次最多 30 字；免费机会用完后，只有注册用户可使用一次性使用码继续生成。

**Architecture:** 在 API 层作为唯一准入点实现 TTS 权益校验，前端只做状态提示与输入辅助，不展示使用码库存、明文码或消费过程。数据库新增免费次数状态与模块化 `UsageCode` 表，使用码以哈希形式存储并在创建 TTS 任务的同一个事务内原子作废，避免并发重复消费。

**Tech Stack:** Next.js 15、React 19、TypeScript、Hono、Bun、Prisma 7、PostgreSQL、MinIO。

---

## Scope Check

本需求只涉及 TTS 生成准入、使用码库存、前端 TTS 表单与部署/数据字典同步，属于同一业务子系统。不拆成多个独立计划。

明确范围：

- 匿名用户可免费生成 1 次，免费生成时 `text.trim().length <= 30`，之后只能提示注册/登录。
- 注册用户可免费生成 1 次，免费生成时 `text.trim().length <= 30`，之后每次生成都必须提交一个有效使用码。
- 30 字限制绑定“免费生成机会”；使用码生成不在本计划中额外展示“免费 30 字”文案，仍复用当前 TTS 正常文本上限（现有实现为 500 字），由后端统一校验。
- 匿名用户不能提交、校验或消费使用码；即使请求体带 `usageCode`，后端也必须拒绝。
- 使用码按模块区分，当前只规划一个模块：`VOICE_TO_TEXT`（用户口径“语音转文字模块”）；后续新增模块时复用同一张 `UsageCode` 表和后台管理能力。
- 使用码只在需要注册用户输入时作为输入项出现，不在使用过程中展示库存、明文码、哈希、消费记录或后台信息。
- 使用码为 6 位 `0-9A-Za-z` 组合，由服务端批量生成，数据库只保存哈希，生成脚本一次性打印明文。
- 使用码一旦通过校验并成功创建 TTS 任务即作废；如果第三方合成失败，使用码仍保持已消费，运营侧可补发新码。这是为了防止并发重复消费和重放。
- 使用码后续需要支持后台查询和后台生成；本计划先把数据结构、模块字段和服务端 helper 设计为后台可复用，不在本次实现完整管理 UI。

非目标：

- 不做支付、套餐、会员。
- 不做完整后台管理界面，但数据模型与接口边界必须支持后续后台查询和生成使用码。
- 不改变建声、录音、声纹作废逻辑。
- 不允许匿名用户输入或消费使用码继续生成。
- 不引入数据库外键，继续遵循当前 `relationMode = "prisma"`。

## Strategy Gate

推荐方案：`User.freeTtsUsedAt` / `AnonymousUser.freeTtsUsedAt` + 带 `module` 字段的 `UsageCode` 单表 + TTS 创建事务内消费。

备选方案 A：只根据历史 `TtsJob` 计数判断免费次数。

- 优点：少加字段。
- 缺点：每次请求都需要查历史；匿名迁移后容易出现重复免费；旧失败任务、历史清理会影响权益判断。

备选方案 B：建立通用 `EntitlementLedger` 权益流水。

- 优点：扩展性最好，可支持套餐、补偿、退款、跨模块消耗流水。
- 缺点：当前 MVP 过度设计，需要更多状态机和后台能力。

选择推荐方案的原因：当前只需要“一次免费 + 模块化一次性码”，两个时间戳和一张带模块字段的码表足够表达业务事实，事务边界清晰，迁移成本低，并能为后续后台查询/生成与新增模块保留明确扩展点。

  ```ts
  import { createHmac, randomInt } from "node:crypto";

  const USAGE_CODE_CHARSET = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz";
  const USAGE_CODE_LENGTH = 6;
  const USAGE_CODE_PATTERN = /^[0-9A-Za-z]{6}$/;

  export function generateUsageCode() {
    let code = "";

    for (let index = 0; index < USAGE_CODE_LENGTH; index += 1) {
      code += USAGE_CODE_CHARSET[randomInt(USAGE_CODE_CHARSET.length)];
    }

    return code;
  }

  export function normalizeUsageCodeInput(code: string) {
    return code.trim();
  }

  export function isUsageCodeFormatValid(code: string) {
    return USAGE_CODE_PATTERN.test(normalizeUsageCodeInput(code));
  }

  export function hashUsageCode(code: string, module: string, secret: string) {
    return createHmac("sha256", secret).update(`${module}:${normalizeUsageCodeInput(code)}`).digest("hex");
  }

  export function buildUsageCodePreview(code: string) {
    const normalized = normalizeUsageCodeInput(code);
    return `${normalized.slice(0, 2)}****`;
  }
  ```

  import { loadConfig, buildDatabaseUrl } from "../lib/config";
  import { initPrisma, prisma } from "../lib/prisma";
  import { buildUsageCodePreview, generateUsageCode, hashUsageCode } from "../lib/usage-code";

  const DEFAULT_USAGE_CODE_MODULE = "VOICE_TO_TEXT";

  function readCount() {
    const index = process.argv.indexOf("--count");
    const raw = index >= 0 ? process.argv[index + 1] : "50";
    const count = Number(raw);

    if (!Number.isInteger(count) || count <= 0 || count > 10000) {
      throw new Error("--count 必须是 1 到 10000 之间的整数");
    }

    return count;
  }

  function readModule() {
    const index = process.argv.indexOf("--module");
    const module = index >= 0 ? process.argv[index + 1] : DEFAULT_USAGE_CODE_MODULE;

    if (module !== DEFAULT_USAGE_CODE_MODULE) {
      throw new Error("当前仅支持 VOICE_TO_TEXT 模块");
    }

    return module;
  }

  const cfg = loadConfig();
  initPrisma(buildDatabaseUrl(cfg.database));

  const count = readCount();
  const module = readModule();
  const generated: string[] = [];

  while (generated.length < count) {
    const code = generateUsageCode();

    try {
      await prisma.usageCode.create({
        data: {
          module,
          codeHash: hashUsageCode(code, module, cfg.usageCode.hashSecret),
          codePreview: buildUsageCodePreview(code),
        },
      });
      generated.push(code);
    } catch (error) {
      if (error instanceof Error && error.message.includes("Unique constraint")) {
        continue;
      }
      throw error;
    }
  }

  console.log(generated.join("\n"));
  await prisma.$disconnect();
