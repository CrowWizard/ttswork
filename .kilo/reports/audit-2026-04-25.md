## Keyboard Focus Order Audit

### 短信登录路径
手机号输入框 → 登录方式 tablist 中的"短信登录"tab → 验证码输入框 → 发送验证码按钮 → 短信登录按钮 → 注册链接

### 密码登录路径
手机号输入框 → 登录方式 tablist 中的"密码登录"tab → 密码输入框 → 密码登录按钮 → 注册链接

### 工作台路径（登录后）
工作区头部（用户名/退出） → 录音面板（建声录音） → TTS 面板（文本转语音）

### 设置页路径
账号信息区域 → 密码管理 tablist → 密码输入框 → 确认密码输入框 → 验证码输入框 → 发送验证码按钮 → 提交按钮

### 键盘操作验证
- Tab/Shift+Tab：线性导航，无正数 tabIndex
- 登录模式切换：ArrowLeft/Right/Up/Down 切换，Home/End 跳转
- 录音控制：Space/Enter 开始，再次按键结束（键盘单键切换模式）
- 隐藏面板不保留可聚焦控件
